El Laboratorio del semáforo consistió en crear una aplicación simple utilizando Jetpack Compose para simular el funcionamiento de un semáforo real. 
El objetivo principal fue practicar el manejo de estados y el uso de LaunchedEffect para controlar cambios automáticos en la UI mediante corrutinas. 
El semáforo está compuesto por tres luces (roja, amarilla y verde) representadas por círculos que cambian de color según el estado actual. 
La aplicación cicla de manera infinita entre los estados: rojo por 2 segundos, verde por 2 segundos y amarillo por 1 segundo, sin necesidad de interacción del usuario. 
A través de este proyecto se reforzó el uso de state management, estructuras como enum, manejo de tiempos con delay, composición de interfaces con Column y Box, 
y la creación de una interfaz limpia y centrada visualmente.
