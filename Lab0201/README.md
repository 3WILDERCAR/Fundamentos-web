El Laboratorio 1 consistió en crear una aplicación en Android con Jetpack Compose que simula un generador de atributos para un personaje de estilo RPG usando tres dados
independientes. Cada dado representa una estadística (VIT, DEX y WIS) y cuenta con su propio botón para lanzar, animación individual, colores dinámicos según el valor 
obtenido y una interfaz ordenada con tarjetas Material Design. El sistema suma automáticamente los tres resultados, muestra el total y genera un mensaje evaluando 
el rendimiento del personaje según ese puntaje. El proyecto permitió practicar conceptos fundamentales como state management, recomposición, corrutinas para 
animación, diseño UI con Rows, Columns y Cards, y la creación de componentes reutilizables como DiceRow.
